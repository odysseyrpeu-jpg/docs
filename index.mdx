## Get started in three steps

<Steps>
  <Step title="Install">
    Install the Hiszpan Scripts CLI globally using your preferred package manager:

    ```bash
    npm install -g hiszpan-scripts
    ```
  </Step>
  <Step title="Initialize">
    Run the `init` command inside your project directory to generate a configuration file:

    ```bash
    hiszpan init
    ```
  </Step>
  <Step title="Run your first script">
    Pick a script from the library and execute it with the `run` command:

    ```bash
    hiszpan run hello-world
    ```
  </Step>
</Steps>

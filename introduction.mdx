Hiszpan Scripts is a collection of ready-to-use scripts and a simple command-line interface designed to eliminate the repetitive work that slows developers down. Whether you're bootstrapping a new project, automating a CI step, or stitching together a multi-stage workflow, Hiszpan Scripts gives you a consistent, composable toolkit you can drop into any codebase and use immediately. It's built for individual developers and teams alike — no complex setup, no vendor lock-in, just scripts that work.

## What you can do

<CardGroup cols={2}>
  <Card title="Run scripts instantly" icon="play">
    Execute any script from the library with a single CLI command — no boilerplate, no configuration required to get started.
  </Card>
  <Card title="Configure behavior" icon="sliders">
    Tailor every script to your environment using a simple `hiszpan.config.js` file that lives alongside your project code.
  </Card>
  <Card title="Automate workflows" icon="arrows-spin">
    Chain multiple scripts together into named workflows and trigger them from the CLI, a CI pipeline, or a cron job.
  </Card>
  <Card title="Extend with custom scripts" icon="puzzle-piece">
    Write your own scripts using the Hiszpan Scripts API and share them across projects or publish them for the community.
  </Card>
</CardGroup>

## Key features

- **Zero-config defaults** — every script ships with sensible defaults so you can run it immediately without touching a config file.
- **Composable workflows** — combine scripts into reusable, named pipelines that you can invoke with a single command.
- **Cross-platform support** — Hiszpan Scripts runs on macOS, Linux, and Windows without platform-specific workarounds.
- **First-class TypeScript support** — write custom scripts in TypeScript with full type definitions included out of the box.
- **Minimal dependencies** — the core CLI has a small, audited dependency footprint to keep your project lean and secure.

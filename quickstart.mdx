This guide walks you through everything you need to get up and running with Hiszpan Scripts. By the end, you'll have Hiszpan Scripts installed in your project, your workspace initialized, and your first automation script successfully executed.

<Steps>
  <Step title="Install Hiszpan Scripts">
    Add Hiszpan Scripts to your project using your preferred package manager.

    <CodeGroup>
      ```bash npm
      npm install --save-dev hiszpan-scripts
      ```

      ```bash yarn
      yarn add --dev hiszpan-scripts
      ```

      ```bash pnpm
      pnpm add --save-dev hiszpan-scripts
      ```
    </CodeGroup>
  </Step>

  <Step title="Initialize Your Project">
    Run the `init` command from the root of your project to set up Hiszpan Scripts:

    ```bash
    script init
    ```

    This creates a `hiszpan.config.json` file in your project root. The config file is where you define your scripts, set environment-specific options, and customize runtime behavior. Open it to review the defaults and adjust any settings to fit your project.

    ```json hiszpan.config.json
    {
      "version": "1.0.0",
      "scripts": {},
      "options": {
        "logLevel": "info",
        "timeout": 30000
      }
    }
    ```
  </Step>

  <Step title="List Available Scripts">
    Check which scripts are available in your workspace at any time:

    ```bash
    script list
    ```

    You'll see output similar to the following, showing each script's name and a short description:

    ```
    Available scripts:

      hello-world       A simple greeting script to verify your setup
      build             Compile and bundle your project for production
      lint              Run the linter across all source files
      test              Execute the full test suite

    Run `script run <name>` to execute any script.
    ```
  </Step>

  <Step title="Run Your First Script">
    Execute the built-in `hello-world` script to confirm everything is working:

    ```bash
    script run hello-world
    ```

    If your installation is successful, you'll see the following output:

    ```
    ✔ Running script: hello-world
    ──────────────────────────────
    Hello from Hiszpan Scripts! 🎉
    Your setup is working correctly.
    ──────────────────────────────
    ✔ Script completed in 0.12s
    ```
  </Step>
</Steps>
